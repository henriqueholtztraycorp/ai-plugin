Repositório oficial da Wake Commerce com plugins, agentes e skills de IA para apoiar agências na implantação de layouts de lojas usando a tecnologia Wake Storefront.

Este projeto reúne agentes especializados que podem ser utilizados em ferramentas como Cursor, Claude e outras plataformas de IA compatíveis, com foco em:

- Revisão técnica de layout
- Análise de performance
- Auditoria técnica de SEO

Nosso objetivo é aumentar produtividade, padronizar entregas e reduzir retrabalho em projetos de storefront.

Outras documentações: [https://wakecommerce.readme.io/docs/visao-geral](https://wakecommerce.readme.io/docs/storefront-ssr-visao-geral)
